k-vim配置 github

作用: 标签展示以及快速跳转(增强vim的书签功能)

github: signature

安装

Bundle "kshenoy/vim-signature"

使用

使用默认快捷键

m[a-zA-Z]   打标签
'[a-zA-Z]   跳转到标签位置

'.          最后一次变更的地方
''          跳回来的地方(最近两个位置跳转)

m<space>    去除所有标签

最终配置

Bundle "kshenoy/vim-signature"
