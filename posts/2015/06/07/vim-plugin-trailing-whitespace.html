k-vim配置 github

没有什么太多需要讲的, 功能+一个快捷键

代码洁癖/强迫症必备

作用: 高亮行末空格(标红), 也可以一键去除文件中所有行行尾空格

github: vim-trailing-whitespace

安装

Bundle 'bronson/vim-trailing-whitespace'

使用

被动技能: 自动标记行尾的空格

主动技能: 绑定<leader><space>为快捷键, 一键去除所有行尾空格 (k-vim中为,空格)

最终配置

Bundle 'bronson/vim-trailing-whitespace'
map <leader><space> :FixWhitespace<cr>
