k-vim配置 github

第一个需要手动安装的插件, 其他的插件通过这个来进行管理

作用: 管理其他所有插件(安装/更新/移除)

github:vundle

安装

1
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vi

使用

在.vimrc中加入/修改/删除自己需要的插件配置

1
Bundle 'scrooloose/syntastic'

然后, 在命令行模式下运行

1
2
3
:BundleInstall     install 安装配置的插件
:BundleInstall!    update  更新
:BundleClean       remove plugin not in list 删除本地无用插件

其他

  1. 同类插件 pathogen, neobundle

  2. 依旧是同类插件, vim-plug, 不过这个支持并行安装插件(目测如果是第一次配置机器会快很多....)
