k-vim配置 github

主题, 在k-vim中, 默认配置了两个

1. solarize

github: solarized

配置

Bundle 'altercation/vim-colors-solarized'
let g:solarized_termtrans=1
let g:solarized_contrast="normal"
let g:solarized_visibility="normal"

2. molokai

github: molokai

配置

Bundle 'tomasr/molokai'
" monokai原始背景色
let g:molokai_original = 1

启用:

Bundle之后安装, 仅仅是安装, 要启用哪个主题, 需要在vimrc中显式指定

set background=dark
set t_Co=256
colorscheme solarized
" colorscheme molokai
" colorscheme desert

建议

  1. 有包含大部分主题的插件, 但是就个人而言, 够用就行, 所以k-vim中只保留了两个, 曾经习惯monokai, 后来转到solarized, 自己喜欢的才是最好的
